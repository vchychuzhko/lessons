HTML structure

            <body>
              <div>
                <span>Score: <span>0<span></span>
                <button type="button">Click</button>
              </div>
            </body>

Page to full height

            html {
              min-height: 100%;
            }
            body {
              margin: 0;
              min-height: 100%;
            }

How to center a div?*

            div {
              align-items: center;
              display: flex;
              flex-direction: column; /* to align items vertically */
              justify-content: center;
            }

*center child elements inside div or any other element

ID selector

            <span id="score">0</span>
            ...
            <script type="module">
              const scoreResult = document.getElementById('score');
              ...
            </script>

Listening to click

            <button type="button" id="clicker"></button>
            ...
            <script type="module">
              const clickerButton = document.getElementById('clicker');
              ...
              clickerButton.addEventListener('click', () => {});
            </script>

Variables

            let clicksCount = 0; // initial value
             
            clickerButton.addEventListener('click', () => {
              clicksCount = clicksCount + 1; // increasing counter by 1
              ...
            });

Let vs Const

Let

Variable can be changed

                    let counter = 0;
                    ...
                    counter = counter + 1;

Const

Variable cannot* be changed

                    const button = docume...;
                    ...
                    button = anotherButton;

*it's not supposed to be changed, only used

Output data

            const scoreResult = document.getElementById('score');
            ...
            clickerButton.addEventListener('click', () => {
              clicksCount = clicksCount + 1;
             
              scoreResult.textContent = clicksCount;
            });

Separate css/js files

    CSS

  • Create styles.css file in the same folder
  • Cut and paste <style> tag content to it
  • Add <link> with respective href

    JS

  • Create script.js file in the same folder
  • Cut and paste <script> tag content to it
  • Add <script> with respective src

Separate css/js files

            <head>
              ...
              <link rel="stylesheet" href="styles.css">
              ...
              <script type="module" src="script.js" defer></script>
            </head>
