Git. Setup

Brought to you by Vladyslav Chychuzhko

Steps

  1. Install Git
  2. Create Github account
  3. Generate and add SSH-key
  4. Config local account information
  5. You're good to go!

We Need

  1. Windows PC
  2. Installed Git
  3. Github account

Generate SSH Key

It is kind of computer's fingerprint that allows VCS* to identify you.

  1. Open "Git Bash"
  2. Run ssh-keygen
  3. "Enter" to all questions

*Version Control System - class of systems responsible for managing changes

Add SSH Key

  1. Open "Git Bash"
  2. Show public key with cat ~/.ssh/id_rsa.pub command
  3. Add it to your Github account under "SSH-keys" section in Settings

Local Account Configuration

In Git Bash: 
            git config --global user.name "Vladyslav Chychuzhko"
            git config --global user.email "vladch@mail.com"

These name and email will be used to sign your commits, pointing to the respective Github account if any.

You're good to go!
