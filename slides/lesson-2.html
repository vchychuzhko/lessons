IDE - Integrated Development Environment

Start with VS Code

  1. Open your project folder in VS Code (or create a new one)
  2. Create or open needed html file
  3. Select "Open with Live Server" by the right click
  4. Start writing your code!

Separate css/js files

  1. HTML document size is reduced
  2. Browser cache
  3. No logic/styles/structure overlay
  4. Reusability of components
  5. Files organization and preprocessing

Separate css/js files

    CSS

  • Create styles.css file
  • Move <style> content to it
  • Add <link> with respective href

    JS

  • Create script.js file
  • Move <script> content to it
  • Add <script> with respective src

Separate css/js files

            <head>
              ...
              <link rel="stylesheet" href="css/styles.css"/>
              ...
              <script type="module" src="js/script.js"></script>
            </head>

Git introduction

Version Control System is a class of systems responsible for managing changes.

  • Traks changes and "packs" them into commits
  • Each commit has its date and author
  • Codebase and all changes (commits) are stored on remote repository*

*remote folder to store your project

Start with Git

  1. Install Git
  2. Log in or sign up on Github
  3. Create a repository
  4. Init local repository ("Source control" tab in VS Code)
  5. Assign your local repository (... > Remote > "Add Remote")
  6. Start making commits!

Download cookie image

  1. Inspect the cookie button (right click > "Inspect")
  2. Find style responsible for image link (background-image)
  3. Open it in a new tab and download
  4. Place image in project folder
  5. Update your style according to new relative path*
            background-image: url('../images/cookie.png');

*Github Pages requires all paths to be relative

Add reset button

Remember Functional Minimum?

  1. Start with HTML structure
  2. Style it!
  3. Add some magic logic

Add reset button

            <div class="clicker">
              ...
              <button class="clicker__reset" type="button" data-reset>
                Reset
              </button>
            </div>

Add reset button styles

            .clicker__reset {
              position: absolute;
              right: 10px;
              top: 10px;
            }

Add reset button logic

            const resetButton = document.querySelector('[data-reset]');
            ...
            resetButton.addEventListener('click', () => {
              clicksCount = 0;
             
              result.innerText = clicksCount;
            });

Keep it DRY*

                const setCount = (count) => {
                  clicksCount = count;
                 
                  result.innerText = count;
                };
            
                clickButton.addEventListener('click', () => {
                  setCount(clicksCount + 1);
                });
                 
                resetButton.addEventListener('click', () => {
                  setCount(0);
                });

*Don't Repeat Yourself

Publish the project

  1. Have index.html file in your project root
  2. Go to your project repository page
  3. Open "Settings" tab, then "Pages"
  4. Select "master" (or "main") branch as target in "Source"
  5. Save and go to your-account.github.io/your-repo
